задача

Поставка, монтаж и запуск оборудования для зала «Республиканского центра тылового обеспечения» Министерства по чрезвычайным ситуациям Республики Беларусь.

процесс

Команда IBT успешно завершила реализацию очередного проекта по оснащению конференц-зала мультимедийной системой. На этот раз наши специалисты выполнили поставку, монтаж и запуск оборудования для зала «Республиканского центра тылового обеспечения» Министерства по чрезвычайным ситуациям Республики Беларусь.

оборудование

Для отображения информации были применены профессиональная ЖК панель Nec, а также мониторы Dell с поддержкой разрешения FULL HD подключаемые через специальные мультимедийные модули установленные в стол. Для коммутации было использовано оборудование SHINYBOW и Gembird. Инсталлированная система предназначена для визуального сопровождения различных мероприятий (презентаций, докладов, совещаний и т.п.).
